# Split Expense App

iOS application built with Swift for splitting expenses between people.  
Easily track shared costs, add participants, and manage balances.

## Features

- Add and manage people in a group
- Create and edit expenses
- Split expenses between participants
- Track who owes whom
- Simple and clean UI for iOS
